# RPGMaker VX ACE 工程搜索工具

此工具旨在帮助用户快速搜索 RPGMaker VX ACE 工程文件中的各种资源，例如物品、武器、防具、敌群、公共事件等。通过该工具，您可以轻松地在庞大的工程文件中找到所需的资源。

## 功能

- **物品搜索**
- **武器搜索**
- **护甲搜索**
- **敌群搜索**
- **角色搜索**
- **开关搜索**
- **变量搜索**
- **公共事件搜索**
